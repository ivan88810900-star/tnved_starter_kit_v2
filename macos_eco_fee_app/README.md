# macOS Eco Fee Calculator (РФ, 2026–2027)

Небольшое настольное приложение для macOS (Python + Tkinter), которое считает экологический сбор по выбранной группе и году и показывает раскладку расчёта.

## Данные ставок (2026–2027)

Ставки (руб./тонну) зашиты в `data/eco_fee_rates_2026_2027.json` и собраны из приложения к **ПП РФ от 01.08.2024 № 1041** (ред. от **25.12.2025**, действует с **01.01.2026**).

- Источник (онлайн-текст с таблицей): `https://normativ.kontur.ru/document?moduleId=1&documentId=503984`

## Формула, реализованная в приложении

Приложение считает **для сценария “утилизация не обеспечена”** (упрощённый сценарий):

- Масса к оплате = `масса_кг`
- Экосбор = `ставка_руб_за_тонну × (масса_к оплате_кг / 1000) × коэффициент_с_2026`

**Важно:** коэффициент берётся автоматически из приложения к ПП РФ №1041 (“с 2026 года”). Нормативы утилизации в этой версии **не используются**.

## Запуск (из исходников)

```bash
cd macos_eco_fee_app
python3 -m venv .venv
source .venv/bin/activate
python3 src/eco_fee_app.py
```

## Сборка `.app` (опционально, через py2app)

1) Установите зависимости:

```bash
cd macos_eco_fee_app
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

2) Соберите:

```bash
python setup.py py2app
```

Готовое приложение появится в `dist/`.
