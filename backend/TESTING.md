## 🔍 Типичные проблемы

### 1. Ошибка импорта
```bash
# Решение: установите зависимости
pip install -r requirements.txt
```

### 2. Ошибка базы данных
```bash
# Решение: проверьте SQLite
rm test.db
pytest tests/
```

### 3. Ошибка файлов
```bash
# Решение: очистите временные файлы
find . -name "*.xlsx" -delete
pytest tests/
```

### 4. Ошибка переменных окружения
```bash
# Решение: сбросьте переменные
unset AI_OFFLINE_MODE ALLOW_EXTERNAL_AI
pytest tests/
```
