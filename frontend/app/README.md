# TN VED Pro - Flutter App

Полноценное Flutter приложение для классификации товаров по ТН ВЭД с использованием ИИ.

## 🚀 Особенности

- **Архитектура**: Riverpod + GoRouter
- **Дизайн**: Темная тема с брендовыми цветами
- **Страницы**: Классификация, Пакетная обработка, Поиск, История
- **Анимации**: Плавные переходы и Hero-анимации
- **Сервисы**: Dio с retry логикой

## 🎨 Дизайн

### Цветовая схема
- **Основной синий**: #0F62FE
- **Графит**: #1B1F23  
- **Золото**: #C6A667 (для highlight)
- **Темный фон**: #0A0A0A
- **Поверхности**: #1A1A1A, #2A2A2A

### Компоненты
- Карточки с тенями и скругленными углами
- Анимированные кнопки и переходы
- Иконки Lucide Icons
- Типографика Inter

## 📱 Страницы

### 1. Главная (Home)
- 4 основные кнопки действий
- Статистика системы
- Брендированный AppBar с логотипом

### 2. Классификация (Classify)
- Текстовое описание товара
- Загрузка изображения
- Чипы-подсказки для характеристик
- Результат с карточкой кода, тарифами, мерами
- Кнопка сохранения в аудит

### 3. Пакетная обработка (Batch)
- Выбор Excel файла
- Прогресс-бар обработки
- Скачивание результата
- Поддержка до 1000 товаров

### 4. Поиск (Explorer)
- Поиск по коду или описанию
- Двухпанельный интерфейс
- Детальная информация о коде
- Тарифы, НТМ, примечания

### 5. История (History)
- Список всех классификаций
- Фильтрация и поиск
- Статистика по точности
- Детальный просмотр

## 🚀 Запуск

### Требования
- Flutter SDK 3.0+
- Dart 3.0+
- Android Studio / Xcode

### Установка
```bash
# Установка зависимостей
flutter pub get

# Генерация кода
flutter packages pub run build_runner build

# Запуск
flutter run
```

### Конфигурация
Создайте файл `.env` в корне проекта:
```env
# API Configuration
API_BASE_URL=http://127.0.0.1:8000
API_TIMEOUT=30000

# AI Configuration  
AI_PROVIDER=openai
AI_OFFLINE_MODE=false
```
