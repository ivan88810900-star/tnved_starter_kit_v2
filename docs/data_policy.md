# Политика данных

- Используем только официальные источники (ЕЭК/ЕАЭС, ФТС России и иные гос-порталы).
- Никакого «живого» веб-скрейпа: только вручную скачанные файлы (HTML/CSV/XML/PDF/ZIP) с официальных порталов.
- Каждая загрузка фиксируется: источник, URL, версия, дата публикации, хеш (SHA256), лицензия/правовой статус.
- Вся обработка — локально. Без отправки материалов внешним ИИ без явного разрешения в .env.
