## 🚀 fetch_tariff_eec.py

### Назначение
Автоматическая загрузка тарифных данных с официального сайта Евразийской экономической комиссии.

### Функциональность
- ✅ **Парсинг страницы** - поиск ссылок на PDF файлы
- ✅ **Загрузка файлов** - с повторными попытками и таймаутами
- ✅ **Проверка целостности** - SHA256 хеши для всех файлов
- ✅ **Обновление конфигурации** - автоматическое обновление sources.yml
- ✅ **Безопасность** - только разрешенные домены
- ✅ **Пропуск существующих** - не загружает файлы с совпадающими хешами

### Использование

#### Базовое использование
```bash
cd data/ingest/
python fetch_tariff_eec.py
```

#### С параметрами
```bash
# Указать другую страницу
python fetch_tariff_eec.py --url "https://eec.eaeunion.org/other-page/"

# Указать другую версию
python fetch_tariff_eec.py --version "2025-02-01"

# Только найти ссылки, не загружать
python fetch_tariff_eec.py --dry-run
```

#### Проверка без загрузки
```bash
python fetch_tariff_eec.py --dry-run
```

## 🔧 Конфигурация

### Обновление sources.yml
Скрипт автоматически обновляет `data/sources.yml`:

```yaml
datasets:
  tariff_cet:
    name: "Единый таможенный тариф (ЕТТ ЕАЭС)"
    authority: "ЕЭК"
    version: "2025-01-01"
    file_type: "PDF"
    urls:
      - "https://eec.eaeunion.org/documents/tariff_2025.pdf"
      - "https://eec.eaeunion.org/documents/rates_2025.pdf"
    save_as:
      - "tariff_2025.pdf"
      - "rates_2025.pdf"
    checksum:
      - "a1b2c3d4e5f6789..."
      - "b2c3d4e5f6789a1..."
```

### Настройка параметров
```python
class TariffEECFetcher:
    def __init__(self, base_url="https://eec.eaeunion.org/comission/department/catr/ett/"):
        self.base_url = base_url
        self.session.headers.update({"User-Agent": "tnved-pro/1.0"})
        self.version = "2025-01-01"
        self.download_dir = Path("data/raw/tariff_cet") / self.version
```

### Проверка целостности
- **SHA256 хеши** для всех загруженных файлов
- **Автоматический расчет** при загрузке
- **Проверка существующих** файлов перед загрузкой
- **Пропуск файлов** с совпадающими хешами

### Обработка ошибок
- **Таймауты** - 60 секунд на загрузку
- **Повторные попытки** - до 3 попыток с экспоненциальной задержкой
- **Валидация URL** - только разрешенные домены
- **Проверка файлов** - размер, формат, целостность

## 🐛 Устранение проблем

### Ошибка загрузки страницы
```
❌ Ошибка загрузки страницы: 404 Not Found
```
**Решение**: Проверьте URL в параметре `--url`

### Ошибка парсинга PDF ссылок
```
📄 Найдено 0 PDF файлов
```
**Решение**: Проверьте структуру HTML страницы, возможно изменились селекторы

### Ошибка загрузки файлов
```
❌ Не удалось загрузить https://example.com/file.pdf после 3 попыток
```
**Решение**: Проверьте доступность файла и интернет-соединение

### Ошибка сохранения конфигурации
```
❌ Ошибка сохранения data/sources.yml: Permission denied
```
**Решение**: Проверьте права доступа к файлу и директории

## 📊 Метрики

### Производительность
- **Время загрузки страницы**: 1-3 секунды
- **Время поиска ссылок**: <1 секунды
- **Время загрузки файла**: 5-30 секунд (зависит от размера)
- **Время расчета SHA256**: <1 секунды на файл

### Размеры файлов
- **Типичный PDF**: 1-10 MB
- **Общий размер**: 10-100 MB (зависит от количества файлов)
- **Место на диске**: +20% для временных файлов
