# TWS.BY — выгрузка ТН ВЭД и ставок ЕТТ

**Сайт:** [Скачать коды ТНВЭД в Excel](https://www.tws.by/tws/tnved/download) (ООО «ФСТ», система TWS).

## Что даёт источник

- Excel-файл с действующими **кодами ТН ВЭД ЕАЭС**, **наименованиями** и **ставками ввозной пошлины** (ЕТТ).
- Файл **обновляется ежедневно** (на странице указана актуальность на дату выгрузки).
- Распространяется **безвозмездно**, «как есть», без гарантий (формулировка на сайте).

## Интеграция в CustomsClear

1. Скачайте Excel со страницы TWS (кнопка на [tws/tnved/download](https://www.tws.by/tws/tnved/download)).
2. В backend: **`POST /api/sources/import`**, передайте файл `.xlsx` (multipart, поле `file`).
3. Импортёр пытается распознать колонки по заголовкам (рус/англ) или взять **первые столбцы** как код / наименование / пошлина.

После импорта калькулятор и поиск по коду используют данные из вашей БД так же, как после CSV/JSON.

## API TWS

На той же странице указано, что для **автообновления справочника** можно подключить **API системы TWS** и писать на **support@tws.by**. Отдельный публичный URL выгрузки в репозиторий не зашит — подключение по согласованию с TWS.

## Приоритет источников (рекомендация)

1. **Локальная БД** после импорта с TWS / ЕЭК / своего CSV.  
2. **Альта-Софт** — опционально, как **запасной** внешний подсказчик по классификации (`docs/integration/alta_*.md`), не замена нормативной базы в приложении.
