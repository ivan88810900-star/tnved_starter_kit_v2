# Классификация по ТН ВЭД ЕАЭС — напоминания для декларанта

Десятизначный код ТН ВЭД определяет ставки и нетарифные меры. При сомнениях сверяйтесь с официальной номенклатурой и пояснениями к разделам.

Бытовая техника часто относится к группе 85 (электрические машины и оборудование). Текстиль и одежда — разделы 50–63. Продукты питания — разделы 01–24 с особыми требованиями ветеринарии и санитарии.

Комбинированные товары классифицируют по материалу, дающему существенные свойства, или по последней операции из правила общей интерпретации 3(b).

Код из ИИ или из внутреннего журнала подсказок требует проверки по описанию партии и технической документации.
