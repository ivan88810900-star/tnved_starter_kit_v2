# Источники для RAG (CustomsClear)

Подключение: в `.env` бэкенда задайте путь от каталога **`customs-clear/backend`**:

```bash
RAG_DOCS_DIR=../docs/rag_sources
```

Файлы **`.md`** и **`.txt`** (и **`.pdf`**) индексируются при запросах к **copilot** / **batch**: токены + окна, опционально `RAG_USE_TFIDF=true`.

Содержимое здесь — **общеобразовательные напоминания**, не замена ТН ВЭД ЕАЭС, ЕТТ и актам. Дополняйте своими внутренними инструкциями компании.
