    Классификационные решения ТН ВЭД, предварительные решения, решения о классификации товаров

    Предварительное решение принимается таможенным органом того государства - члена ЕАЭС, в котором будет осуществляться выпуск товаров. Предварительное решение является обязательным при декларировании товаров на территории государства - члена ЕАЭС, таможенный орган которого принял предварительное решение.

    Предварительное решение принимается на каждое наименование товара, включающее определенную марку, модель, артикул и модификацию.

    Порядок принятия предварительного решения по классификации товаров по Товарной номенклатуре внешнеэкономической деятельности определяется Таможенным кодексом ЕАЭС, а в части, не определенной указанными статьями, - законодательством государств - членов ЕАЭС.

    База "Предварительные решения" содержит следующие данные: код товара, описание товара.

    Поиск информации о предварительных решениях осуществляется по коду товара и описанию товара.
