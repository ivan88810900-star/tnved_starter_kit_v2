# VISION.md — Продуктовая цель Tariff

> Статус: **постоянный**. Это north star для всех AI-агентов и инженерных решений.

---

## Одна фраза

**Tariff помогает пользователю правильно классифицировать товар для таможенного оформления — а не просто показывать коды ТН ВЭД.**

---

## Что это значит на практике

Пользователь приходит не за справочником ради справочника. Он приходит с вопросом:

> «Какой код выбрать для *этого* товара, и что потом потребуется для ввоза?»

Правильный ответ включает:

1. **Классификацию** — код ТН ВЭД с обоснованием, почему именно он, а не соседний.
2. **Контекст** — семантические группы («лососевые», «камбалообразные»), а не плоский список из 45 кодов.
3. **Последствия** — пошлина, НДС, нетарифные меры, сертификаты, лицензии.
4. **Уверенность** — когда код очевиден, когда нужны уточнения, когда нужен брокер/эксперт.

---

## Чего мы **не** делаем

- Не превращаем продукт в «браузер по Excel-выгрузке ТН ВЭД».
- Не показываем пользователю технические артефакты дерева (pad-коды, codeless-узлы, synthetic leaf) без объяснения.
- Не выдаём код без контекста классификации.
- Не подменяем таможенного брокера — но **снимаем с него рутину** там, где это безопасно.

---

## Критерии качества продукта

| Область | Хорошо | Плохо |
|---------|--------|-------|
| Дерево ТН ВЭД | «0302 → лососевые → форель → 0302110000» | 45 плоских кодов под 0302 |
| Поиск | «рыба свежая лосось» → релевантные коды с группами | только exact match по коду |
| Классификация AI | код + обоснование + confidence + уточняющие вопросы | один код без объяснения |
| Платежи | полный профиль: пошлина + НДС + сборы + преференции | только ставка пошлины |
| Нетарифка | ДС/СС/СГР с applicability (definite vs advisory) | сырой дамп TKS-текста |

---

## Архитектурные следствия vision

1. **Semantic Navigation** — отдельный слой смысловых групп поверх структурного дерева. Не заменяет `_build_tree()`, а дополняет UX.
2. **Tree Engine v2** — типизированное дерево для будущей миграции без breaking API.
3. **NTM v2** — official vs legacy isolation; advisory ≠ enforcement.
4. **AI copilot** — RAG + structured JSON, не «угадай код».
